## 社区范围

项目尤其关注来自 AI 基础设施建设者、数据平台团队、分析工程师，以及正在评估可治理 Agent 工作流的产品团队的实践反馈。

高质量社区贡献通常包含：

- 具体的数据工作流、数据源或分析任务。
- 复现步骤、示例 schema、截图、日志或 trace 输出。
- 当前行为、预期行为和产品想法之间的清晰区分。
- 不包含密钥、生产凭据、客户数据或私有业务记录。

## 协作原则

DataFoundry 的文档和示例需要清楚区分已经落地的能力、实验性工作流和路线图想法。社区讨论越基于证据、越可复现，越容易转化为产品改进。
